# PPC-circle-of-life

(1) Open a new terminal / CMD
(2) Move to the root of the project
(3) Start the environment module with ```python -m environment.env```(or using python3)
(4) Restart step (1) and (2) and start the display module with ```python -m display.display``` (or using python3)
(5) Use the textBox to write




Ouvrir une nouvelle fenêtre de terminal / CMD
Aller dans le dossier root du projet
Lancer l’environnement avec python -m environment.env (ou avec python3)
Recommencer les étapes (1) (2) et lancer le display avec python -m display.display (ou avec python3)
Utiliser la textBox pour écrire les commandes et appuyer sur “PRINT” pour les envoyer.
La simulation s’affiche sur le canvas au-dessus.
